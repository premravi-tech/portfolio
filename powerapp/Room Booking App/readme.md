### Unified Room Booking App
This app is used to book rooms in multiple locations of an Organisation.
This app also has 3 roles created - User, Front Desk and Admin. Each of these roles will have its own access permissions.

The purpose of this app is to allow the users of an org to manage their meeting/room bookings in one app.
